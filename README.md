# VPF TEAM — статистика VooPooFamily

Статический сайт с ежемесячными итогами альянса VooPooFamily в Brawl Stars.

## Возможности

- отдельная анимированная главная страница;
- горизонтальная навигация по месяцам;
- архивные итоги за март–июнь 2026 года;
- живая страница июля с данными из Google Таблицы;
- рейтинги клубов, игроков, пушеров, режимов и бойцов;
- адаптивные версии для компьютеров, планшетов и телефонов;
- ссылки на Telegram-канал и техническую поддержку.

## Запуск локально

Можно открыть `index.html` напрямую или запустить локальный сервер:

```bash
python -m http.server 8000
```

Затем открыть `http://localhost:8000`.

## Публикация через GitHub Pages

1. Создайте новый репозиторий на GitHub.
2. Загрузите всё содержимое этой папки в ветку `main`.
3. Откройте **Settings → Pages**.
4. В разделе **Build and deployment** выберите **Deploy from a branch**.
5. Выберите ветку `main` и папку `/ (root)`.
6. Сохраните настройки.

## Google Таблица

Идентификатор таблицы задаётся в начале `script.js` переменной `SHEET_ID`.
Для чтения данных посетителями таблица должна быть доступна по ссылке.

Проект не содержит токенов Telegram, ключей Brawl Stars API или данных сервисного аккаунта Google.

Шрифты хранятся локально в `assets/fonts`. При обновлении сайта эту папку необходимо загружать на GitHub вместе с `index.html`, `style.css` и `script.js`.
